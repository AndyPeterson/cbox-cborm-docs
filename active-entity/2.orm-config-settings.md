# 2.ORM Config Settings

Open your config/ColdBox.cfc and either uncomment or add the following settings:

```javascript
// ORM services, injection, etc
orm = {
    // entity injection
    injection = {
        // enable it
        enabled = true,
        // the include list for injection
        include = "",
        // the exclude list for injection
        exclude = ""
    }
};
```
